# Crypto Layer

**CryptoLayer** - это библиотека, которая не заменяет мессенджеры, а лишает их возможности читать ваши сообщения. Она реализует собственный сетевой стек и шифрует данные до того, как они попадут в мессенджер. 

Проще говоря: для CryptoLayer любой мессенджер - это просто ненадежный "провод", а всё шифрование и гарантия доставки происходят исключительно на вашем устройстве.



## ☝️ Важно

Пользователь имеет фундаментальное право на **приватное и защищённое общение**. Это включает право **самостоятельно применять криптографические средства** для защиты своих сообщений, а также право на конфиденциальность переписки **без несанкционированного доступа третьих сторон**.

Проект исходит из принципа, что безопасное и частное общение - это **базовая цифровая норма, а не привилегия**.
